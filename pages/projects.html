Application de messagerie avec annotation des messages

HTML CSS JavaScript PHP Ratchet WebSocket UX + UI design

Dans le cadre d’un projet de groupe, nous avons développé une application web de messagerie instantanée avec une particularité originale : pour pouvoir continuer à échanger, chaque utilisateur devait annoter les messages envoyés et reçus. Ce mécanisme visait à encourager une réflexion active et une meilleure compréhension des échanges. Nous avons débuté par la conception des maquettes sur Figma, afin de définir l’interface et l’expérience utilisateur.

Techniquement, l’application a été réalisée en HTML, CSS et JavaScript pour la structuration, la mise en forme et la dynamisation des pages, notamment les formulaires. Côté back-end, nous avons utilisé PHP avec une architecture MVC pour gérer la logique applicative et la connexion à la base de données. Pour permettre les échanges en temps réel, nous avons mis en place une communication via WebSocket, en utilisant la bibliothèque PHP Ratchet, ce qui nous a permis d’assurer des conversations fluides et instantanées entre utilisateurs.

Site exposition

HTML CSS JavaScript UX + UI design

Dans le cadre d’un projet scolaire (réalisé en groupe), j’ai créé un site web fictif visant à présenter une exposition imaginée, inspirée de l’exposition réelle À quatre mains consacrée aux œuvres collaboratives de Basquiat et Warhol. J’ai utilisé HTML, CSS et JavaScript pour structurer et styliser le site, intégrer des vidéos, et ajouter une fonctionnalité de traduction des pages en anglais via JavaScript. Les maquettes de l’interface ont été conçues avec Figma afin d’assurer une expérience utilisateur cohérente et visuellement attrayante.

La ferme de la ville

HTML CSS Bootstrap

Dans le cadre d’un projet scolaire, j’ai simulé une collaboration à long terme entre un client et un développeur. En tant que développeur, j’ai été chargé de concevoir un site web autour d’un thème défini par le client. Ce projet m’a permis d’expérimenter une relation de travail itérative : à chaque retour du client, j’ai dû adapter ou modifier certaines parties du site en fonction de ses retours, ce qui m’a appris à gérer les feedbacks et à maintenir un code évolutif. Le site a été développé en HTML, CSS et Bootstrap.

Affichage de formes

Java Bibliotheque Exception (try-catch) Modele UML

En Java, j'ai également eu l'occasion de créer des formes comme des triangles ou des cercles à l'aide de bibliothèques graphiques, dans le but de dessiner une maison entourée de montagnes. Ce projet a marqué mon premier véritable contact avec l'utilisation de bibliothèques externes, un concept essentiel dans le développement moderne

Cette expérience m’a permis de mieux comprendre la logique de la programmation orientée objet en manipulant des classes pour représenter les différentes formes. J’ai aussi découvert l’importance de structurer son code pour le rendre plus lisible et réutilisable. En explorant les possibilités offertes par ces bibliothèques graphiques, j’ai pris conscience de la puissance des outils mis à disposition par la communauté Java, et cela a renforcé mon intérêt pour la conception d’interfaces visuelles interactives.
